# Follow Me - Bash Script Tool

## Bash Script Tool for Searching the Username on list of Social Media Sites

![Actual Look](https://github.com/savjaylade84/followMe/blob/main/followMe2.jpg?raw=true) 

- **This tool get the username then try to search its existent in the list of social media urls**</br>

- **The dependencies needed to install**</br>

	- **wget** `apt install wget`</br>
	- **ruby** `apt install ruby`</br>
	- **lolcat** `gem install lolcat`</br>
	
- **Give permission to the tool to run in the terminal**</br>
`chmod +x followMe`

- **To Run this you need to write this command in the terminal**</br>
`bash followMe`</br>
